Libraries

JQuery 4 years

I use JQuery in most projects I work on, however I'm trying to use it less as more browsers support better native functionality in vanilla Javascript.

For example, the bubbles in the background of this site are animated using native JS that manipulates CSS transforms, which sped up the previous JQuery version by about 5 times.

ReactJS 1 year (Learning)

I've been learning React for about a year now, and I'll likely start using it on a project later this year.

Frameworks

Bootstrap 3 years

I've been using Bootstrap in about three quarters of all projects I work on. It's a very powerful tool that adds responsiveness among other nice features like modals. Generally, I'll customize the distribution package down to only what I'll be using, to avoid bloat.

AngularJS (1.3) 2 years

AngularJS has been one of my favorite frameworks because of it's paradigm. While the learning curve is a bit steeper than some other frameworks, I've found its structure very clean and DRY.

I'm currently learning AngularJS v2
